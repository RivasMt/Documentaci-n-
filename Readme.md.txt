## Descripción general

Este repositorio contiene una implementación en **Python** de un modelo biofísico matemático utilizado para estudiar la actividad eléctrica de neuronas piramidales en la región CA1 del hipocampo. Fue realizado como parte del cumplimiento del servicio social del alumno de física biomédica Rivaldo Hernández Magdaleno para la Universidad Nacional Autónoma de México.

El modelo se basa en una formulación reducida de tipo **Hodgkin–Huxley**, en la que se describen las corrientes iónicas responsables de la dinámica del potencial de membrana neuronal.

El objetivo de este proyecto fue reproducir y extender simulaciones de actividad neuronal mediante modelado computacional. En particular, se implementaron términos adicionales que representan **entrada sináptica**, con el fin de estudiar cómo esta modulación afecta los patrones de disparo neuronal.

Este repositorio incluye:

- Código en **Python**
- **Jupyter Notebooks** con simulaciones
- Figuras generadas a partir de las simulaciones
- Tutoriales que explican paso a paso la implementación del modelo
- Ejemplos de comparación entre actividad eléctrica en **células jóvenes y envejecidas**

Las simulaciones permiten explorar la dinámica neuronal, lo cual busca facilitar el estudio de diferentes escenarios fisiológicos mediante simulaciones computacionales sin necesidad de realizar experimentación directa.
